1️⃣ Menu de Opções com Switch Crie um programa que mostre este menu:

1 - Ver hora atual 2 - Dizer Olá 3 - Sair O usuário escolhe uma opção digitando 1, 2 ou 3. Use switch para mostrar a resposta certa para cada caso.

2️⃣ Somar Números até o Usuário Digitar 0 Peça ao usuário que digite números até que ele digite 0. Some todos os números digitados e mostre o total ao final.

3️⃣ Verificação de Senha com Limite de Tentativas Peça que o usuário digite uma senha. Ele terá 3 tentativas. Se acertar, mostre “Acesso liberado”. Se errar 3 vezes, mostre “Conta bloqueada”.

4️⃣ Contar Palavras de uma Frase Peça que o usuário digite uma frase. Mostre quantas palavras existem na frase (palavras separadas por espaço).

5️⃣ Calcular Média com Quantidade Desconhecida Peça ao usuário que digite várias notas. Quando ele digitar -1, o programa para e mostra a média das notas digitadas.

6️⃣ Cadastro de Nomes Permita que o usuário cadastre nomes. Após digitar cada nome, pergunte se ele quer adicionar mais. Quando terminar, mostre todos os nomes cadastrados.

7️⃣ Procurar Nome na Lista Depois de cadastrar nomes em um array, peça ao usuário um nome. Diga se o nome existe ou não na lista.

8️⃣ Soma dos Números Pares de 1 a 100 Percorra os números de 1 até 100. Some apenas os números pares e mostre o resultado.

9️⃣ Verificar se Palavra é Palíndromo Peça uma palavra. Mostre se ela é um palíndromo (exemplo: “arara”, “radar”, “ovo”).
