Tipos de Saída

1 - O que é um tipo de saída em JavaScript?

Um tipo de saída em JavaScript é uma forma de exibir informações para o usuário.

alert()

Mostra uma janela de alerta no navegador com uma mensagem.

alert("Olá, mundo!");

document.write()

Escreve uma mensagem diretamente na página HTML.

Declarações, Sintaxe e Comentários

1 - Explique o que significa uma declaração em JavaScript.

Uma declaração em JavaScript é uma instrução que diz ao programa o que ele deve fazer. Pode ser usada para criar variáveis, executar funções ou realizar operações.

2 - Qual a importância dos comentários no código?

Variáveis

1 - Qual a diferença entre let, const e var?

let: usada para declarar variáveis que podem ter seu valor alterado depois.

const: usada para declarar variáveis cujo valor não pode ser alterado depois de definido.

var: forma antiga de declarar variáveis em JavaScript. Ainda funciona, mas hoje é menos usada.

2 - Quando usar cada uma dessas declarações?

Use let quando precisar mudar o valor da variável.

Use const quando o valor não vai mudar.

Use var apenas em códigos antigos, para os novos utilize let.

Operadores

1 - O que são operadores aritméticos? Dê 3 exemplos.

Operadores aritméticos são símbolos usados para realizar cálculos matemáticos com valores numéricos.

Exemplos:

Soma (+)

let soma = 5 + 3; // Resultado: 8

Subtração (-)

let subtracao = 10 - 4; // Resultado: 6

Multiplicação (*)

let multiplicacao = 6 * 2; // Resultado: 12

2 - O que são operadores de comparação? Explique com exemplos.

Operadores de comparação são usados para comparar dois valores e retornam um resultado booleano: true ou false.

Exemplos:

Igual (==)

console.log(5 == 5); // true

Maior que (>)

console.log(10 > 7); // true

Menor igual que (<=)

console.log(6 <= 1); // false
